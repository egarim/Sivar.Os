## Expected Benefits

- **Accuracy:** Sub-meter vs ±0.5% error
- **Performance:** 10-100x faster with spatial indexes
- **Features:** Geocoding, reverse geocoding, nearby search
- **Cost:** $0 (all FREE components)

## Key Design Decisions

1. **PostGIS Geography Type:** Use GEOGRAPHY(POINT, 4326) for accurate earth-surface distances
2. **Geocoding Provider:** Nominatim (FREE), can upgrade to Azure Maps later
3. **EF Core Pattern:** Use .Ignore() for GeoLocation columns (same as pgvector)
4. **Map Library:** Leaflet.js with OpenStreetMap tiles (both FREE)
5. **Architecture:** ILocationService abstraction for provider independence
